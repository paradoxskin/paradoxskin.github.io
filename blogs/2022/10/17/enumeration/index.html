Enumeration

用惯了cpp的STL，感觉java的数据结构非常麻烦，不仅前面还写一堆东西，还有各种方法名，所以简单的记一下，这次碰到的enumeration

import java.util.Enumeration

它其实不是一个数据结构，只是一个接口，这个接口只有两个方法，

一个是查看容器中是否还剩东西 hasMoreElements()

一个是取出下一个东西 nextElement()

java.util.Enumeration<String> enum = xxx;
while(enum.hasMoreElements()){
    String x = enum.nextElement();
    xxxx;
}
